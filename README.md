# word-guess-cli

This is a work guess game for the cli.  Guess the Startrek crew members!

Simply navigate to the project directory and run via:
```
node index.js
```
## What you'll see:

- Guesses remaing
- letters guessed correctly
- no repeats allowed
- letter validation
